# 집에서 만드는 진짜 된장찌개

된장찌개는 가장 자주 먹는 찌개인데, 정작 집에서 만들면 어딘가 2% 부족하게 느껴진다는 분들이 많습니다. 맛집처럼 깊고 구수한 된장찌개, 세 가지 핵심 포인트만 지키면 됩니다.

## 재료 (2인분 기준)

| 재료 | 분량 |
|------|------|
| 된장 | 2큰술 |
| 두부 | 1/2모 |
| 호박 | 1/4개 |
| 감자 | 1개 |
| 표고버섯 | 2개 |
| 대파 | 1/2대 |
| 청양고추 | 1개 |
| 다시마 | 10x10cm 1장 |
| 멸치 | 한 줌 |
| 물 | 700ml |

---

## 핵심 포인트 1. 육수를 제대로 내기

시판 육수 대신 직접 육수를 내면 국물이 완전히 달라집니다.

**방법**
1. 냄비에 물 700ml, 다시마, 머리·내장 제거한 멸치를 넣고 냉장에서 30분 우립니다 (찬물에 우리는 것이 핵심!)
2. 중불로 끓기 시작하면 다시마는 바로 건집니다
3. 5분 더 끓인 뒤 멸치를 건져내면 맑고 깊은 육수 완성

> 다시마를 너무 오래 끓이면 쓴맛이 납니다. 끓기 시작하자마자 꺼내는 게 포인트!

---

## 핵심 포인트 2. 된장 선택

시중 된장은 크게 두 가지입니다.

- **공장된장 (혼합장)**: 간이 일정하고 달달한 맛. 초보자에게 적합
- **재래식 된장 (전통장)**: 짜고 깊은 맛. 두 종류를 7:3 비율로 섞으면 최고

처음엔 공장된장으로 연습하고, 익숙해지면 재래식 된장을 섞어보세요.

---

## 핵심 포인트 3. 재료 투입 순서

익는 데 시간이 오래 걸리는 재료부터 넣어야 합니다.

1. 육수에 된장을 풀어 끓입니다
2. 감자를 먼저 넣고 5분
3. 호박, 버섯 투입하고 3분
4. 두부, 대파, 청양고추 넣고 2분 더 끓이면 완성

---

## 완성 팁

- 간이 부족하면 된장이나 국간장으로 조절 (소금 대신)
- 기호에 따라 고춧가루 1/2 작은술을 넣으면 칼칼해집니다
- 새우젓 한 꼬집을 마지막에 넣으면 감칠맛이 올라갑니다
